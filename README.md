# Getting Started

## Installation

### Prerequisites

- [Git](https://git-scm.com/)
- [Bash](https://www.gnu.org/software/bash/)

### Clone

Clone this repository to your local machine

```bash
git clone https://github.com/UTFeight/Scripts
```

### Symlink

It's a good idea to symlink the scripts to a directory in your `$PATH` so that you can run them from anywhere.

to do that you can pass the `--install` flag to the script you use.
